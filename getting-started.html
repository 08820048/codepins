Getting Started with CodePins

Welcome to CodePins, your modern code bookmarking solution for IntelliJ-based IDEs. This guide will help you get started with the essential features.

Step 1: Adding Your First Pin

There are multiple ways to add a pin to your code:

  • Right-click method: Right-click on any line of code and select "Pin This Line" from the context menu.
  • Keyboard shortcut: Place your cursor on any line and press Alt+Shift+P.

You'll be prompted to add an optional note to your pin. This is useful for reminding yourself why this code is important.

Step 2: Viewing Your Pins

All your pins are accessible from the CodePins tool window:

  • Alternatively, press Alt+Shift+T to toggle the CodePins tool window.

The tool window displays all your pins with their file paths, line numbers, and notes.

Step 3: Navigating Between Pins

CodePins makes it easy to jump between important code locations:

  • From the tool window: Double-click any pin to jump to its location.
  • Using keyboard shortcuts:
    • Press Alt+Shift+RIGHT to navigate to the next pin.
    • Press Alt+Shift+LEFT to navigate to the previous pin.

Step 4: Managing Your Pins

Keep your pins organized with these features:

  • Search: Use the search box at the top of the tool window to filter pins by file path or note content.
  • Reorder: Drag and drop pins to arrange them in your preferred order.
  • Edit: Right-click on a pin and select "Edit Note" to update its description.
  • Delete: Right-click on a pin and select "Delete Pin" to remove it, or use the "Clear All" button to remove all pins.

Step 5: Using Tags

Tags help you categorize and filter your pins:

  • When adding or editing a pin, include hashtags in your note (e.g., #important, #bug, #todo).
  • Use the search box to filter pins by tag (e.g., type "#bug" to see only pins with that tag).

Pro Tip: Code Blocks

CodePins supports pinning entire code blocks, not just single lines:

  1. Select multiple lines of code in your editor.
  2. Right-click and select "Pin This Code Block" or use the keyboard shortcut.
  3. The pin will display the line range (e.g., "Lines 10-15") in the tool window.

Pro Tip: Import/Export

Share your pins with team members or between projects:

  1. Click the export icon in the CodePins tool window to save your pins to a file.
  2. Click the import icon to load pins from a previously exported file.

This is great for onboarding new team members or creating standardized navigation points in your codebase.

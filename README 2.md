# CodePins

CodePins 是一个轻量级 IntelliJ 插件，用于在代码中快速打点标记、备注，并在侧边栏集中查看与跳转。

## ✨ 功能特点

- 📌 在代码行上右键 → 添加图钉
- 💬 可输入备注，帮助你记录开发思路或待处理项
- 🧭 在侧边栏中展示所有图钉，双击可跳转定位
- 🔁 添加后立即刷新 UI，无需重启插件

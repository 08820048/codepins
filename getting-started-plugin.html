Getting Started with CodePins

Welcome to CodePins! Follow these simple steps to enhance your coding workflow:

1. Adding Pins

  • Right-click method: Right-click on any code line and select "Pin This Line"
  • Keyboard shortcut: Place cursor on any line and press Alt+Shift+P
  • Add an optional note to remember why this code is important

2. Viewing & Navigating

  • Open the CodePins tool window from the left sidebar or press Alt+Shift+T
  • Double-click any pin to jump to its location
  • Use Alt+Shift+RIGHT and Alt+Shift+LEFT to navigate between pins

3. Managing Pins

  • Search pins by typing in the search box (filters by file path or note content)
  • Drag and drop pins to reorder them
  • Right-click on pins for additional options (edit note, delete, view code)
  • Use tags in notes (e.g., #important, #bug) for better organization

4. Advanced Features

  • Code blocks: Select multiple lines before adding a pin
  • Import/Export: Share pins between projects or team members
  • Persistent storage: Your pins are saved automatically between IDE sessions

For more detailed information, visit our GitHub repository.
